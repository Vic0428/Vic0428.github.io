Efficient Serverless Edge Function Deployment

Overview

In this work, we show that resource slacks in the NFV environment are untapped resources for serverless edge computing. We perform detailed analysis of the benefits and limitations of co-locating serverless functions on NFV-ready servers. We propose NEMO a novel platform that enables efficient serverless edge function deployment in the NFV environment. NEMO can intelligently harvest resource slacks of network functions to warm up the serverless functions in an agile manner. In addition, NEMO can judiciously manage the thread conflict of function pre-warm thread and function invocation thread in a resource-limited environment. We build a prototype of NEMO. Our thorough evaluation show that NEMO can harvest up to 41% resource slacks with guaranteed the throughput of network functions. With the harvested resource slacks, NEMO can achieve about 12.5 ~ 25X performance improvement compared with straightforward co-location. This work demonstrates valuable opportunities to deploy serverless edge computing with great cost-efficiency.
