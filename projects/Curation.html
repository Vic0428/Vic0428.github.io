Scalable Data Curation System

Overview

It is well-known that data scientists spend a significant amount of effort on data preparation. This is especially true for text applications because of the messy nature of text data. In this project, we present TxtWrangler, a scalable text data curation system. TxtWrangler consists of three components, a text normalization and standardization module, an approximate text matching and alignment module, and a module for efficiently operating vector representations. We first show a few use cases of how each of the three modules can help various text learning tasks. Then we describe three featured, novel techniques in TxtWrangler. They are scalable and general programming-by-negative-example methods for text normalization, Allign for all near-duplicate passage detection, and Vecal for all-pair vector operations.
