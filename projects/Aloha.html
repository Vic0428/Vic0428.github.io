Executable Deep Learning Latex Code

Overview

There are increasingly more and more people writing deep learning code. Teachers and students tend to focus more on the math / algorithm behind the model rather than detailed implementation. Researchers publishing their work with code encounter the gap between models described in math/natural language and implementation. So in this project, we proposed a novel language Aloha eliminating the gap between model design and implementation
